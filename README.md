# Stock Market Analysis 

Analisis pergerakan harga saham historis Apple, Google, Tesla, dan Microsoft
menggunakan Python dan SQL.

# Tujuan Analisis
- Mengidentifikasi tren harga jangka pendek dan panjang
- Menghitung indikator teknikal (MA, RSI, MACD)
- Membandingkan performa return antar saham

# Tools
Python · Pandas · Matplotlib · SQLite · Jupyter Notebook

# Dataset
Sumber: Yahoo Finance via `yfinance` | Periode: 2022–2024

# Cara Menjalankan
1. `pip install -r requirements.txt`
2. Buka folder `notebooks/` di Jupyter
3. Jalankan notebook secara berurutan (01 → 02 → 03)

# Hasil Utama
- TSLA memiliki volatilitas tertinggi di periode analisis
- MSFT menunjukkan tren naik paling konsisten
- RSI di bawah 30 pada AAPL terjadi 3x dan diikuti rebound
